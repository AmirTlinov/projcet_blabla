Subagent Playground

Сайт полностью собран через Codex subagents — без ручного редактирования.

Zero Touch

Worktree, sandbox и writer управляются из одной CLI-команды.

Remote Observability

HTTP сервис выдаёт /metrics и streaming logs для каждого агента.

Role Prompts

Каждый subagent получает собственный AGENTS.md и prompts/.

Live Metrics Curl

curl -H "Authorization: Bearer secret" \
  http://127.0.0.1:8075/metrics

CLI аналог: subagents metrics --remote-url http://127.0.0.1:8075 --token secret

Streaming Logs

Используй удалённый follow, чтобы смотреть прогресс субагента без входа на сервер.

subagents logs web-agent --remote-url http://127.0.0.1:8075 \
  --token secret --follow --follow-seconds 15
